| About the pattern                                                 | Ami a mintát illeti                      |
| ----------------------------------------------------------------- | ----------------------------------------- |
| **+** : it's on the right place                             | **+** : benne van és jó helyen    |
| **.** : it's not on the right place, but it's in the word | **.** : benne van, de rossz helyen |
| **-** : it's not in the word                              | **-** : nincs benne sehol           |
